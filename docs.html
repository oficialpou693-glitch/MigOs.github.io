# Sobre o Sistema

O MigOs é uma distribuição Linux independente, construída inteiramente através da metodologia Linux From Scratch (LFS). Diferente de distros baseadas em Debian ou Arch, cada componente do MigOs foi compilado manualmente para garantir o máximo de performance.

O foco principal do projeto é atrair usuários do Zorin OS e do Windows, oferecendo um assistente de boas-vindas inteligente que configura o hardware e explica o funcionamento do terminal de forma didática.

# Suportes e Compatibilidade

A Libraria Universal (miglib)

O MigOs possui um wrapper exclusivo que permite instalar pacotes de qualquer grande ecossistema Linux:

  • APT: Repositórios Debian/Ubuntu.
  • Pacman: Repositórios Arch/AUR.
  • DNF: Repositórios Fedora/RedHat.

Executáveis Windows (.EXE)

Graças a uma camada de compatibilidade de baixo nível baseada no MS-DOS e integrada ao Kernel, o MigOs consegue executar arquivos .exe nativamente sem a necessidade de configurações complexas de Wine por parte do usuário.

# Interface Gráfica

O MigOs é agnóstico de interface, permitindo que o usuário escolha o ambiente que melhor se adapta ao seu hardware:

Hyperland

Tiling Window Manager moderno para máxima produtividade.

GNOME

Interface limpa e intuitiva, ideal para quem vem do Mac.

KDE Plasma

Altamente customizável, ideal para ex-usuários de Windows.

XFCE

Leveza extrema para PCs antigos.
